Biny Задание для AI GDG Hakaton Astana: Интеллектуальная оптимизация мультимодальной логистики
Постановка задачи
Разработать AI-систему оптимизации грузоперевозок для абстрактного логистического хаба с комбинацией наземного и воздушного транспорта. Участники должны самостоятельно сформировать датасет и обучить модели.
Основные требования
Система должна решать:
Выбор оптимального транспорта (авто/авиа/комбинированный)
Прогнозирование времени и стоимости доставки
Оптимизацию маршрутов с учетом множественных факторов
Батчирование заказов для экономии
Самостоятельная подготовка данных
Участники должны сами:
Сгенерировать синтетический датасет с реалистичными параметрами
Использовать открытые источники (OpenStreetMap, публичные API авиарасписаний)
Создать симуляцию логистической сети минимум из 30 узлов
Смоделировать исторические данные за 3-6 месяцев
